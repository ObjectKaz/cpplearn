## 一.转换函数的类型
1.	转换构造函数：从其他类型到类类型 `T`
2.	类型转换函数：从类类型 `T` 到其他类型

## 二.转换构造函数
1.	函数原型 `类名(类型名)`
2.	只有一个形参（类型任意）。
3.	它的主要工作是构造对象，顺带完成类型转换工作。
4.	**既可以被编译器自动调用，也可以被用户强制调用**。

## 三.类型转换函数
1.	本质上是重载强制类型转换运算符。
2.	只能作为 **成员函数** 。
3.	没有参数。
4.	**只能被编译器自动调用**。

## 四.类型转换与二义性
如果类中包含一个或多个类型转换，则必须确保在类类型和目标类型之间只存在唯一一种转换方式。 否则的话，我们编写的代码将很可能会具有二义性。
