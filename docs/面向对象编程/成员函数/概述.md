## 一.成员函数

1.	**与一般函数的区别** 只属于一个类。
2.	类型
   +	在类中定义的成员函数
     +	默认为内联函数
   +	在类外定义的成员函数
     +	需要加上作用域运算符

3.	**存储方式** **成员函数被该类所有对象共享** 每个对象占用的存储空间只是该对象的数据部分占用的存储空间，而不包括成员函数代码占用的存储空间。
4.	**调用方式** 成员函数调用时，接受了一个对象本身的地址，用 this 获取。
