## 二.结构体和类的区别
1.	成员默认权限
	+	如果在结构体的定义中不指定成员访问权限 ，则系统就默认为是 `public`的。
	+	如果在类的定义中不指定成员访问权限 ，则系统就默认为是 `private`的。
2.	继承方式
	+	如果基类不指定成员访问权限 ，则系统就默认为是 `public`的。
	+	如果基类不指定成员访问权限 ，则系统就默认为是 `private`的。
