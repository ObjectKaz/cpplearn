# C++学习笔记

## 内容标注信息

+	`[$]` 拓展内容，教材没有。
+	`[N+]` C++标准N 新出现的内容
+	`[N-]` C++标准N 移除的内容
+	`[N!]` C++标准N 弃用的内容
+	`*` 教材外的章节。

## 编译环境
+	`VC++6.0`
+	`Visual Studio 2019`
+	`gcc 8.1.0(mingw-w64)`
