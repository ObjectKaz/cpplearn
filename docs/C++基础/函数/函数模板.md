## 一.概述
1.	**概念** 建立一个通用函数,其函数类型和形参类型不具体指定,用一个虚拟类型来代表。
2. 	**功能** 解决函数体完全一样,而函数类型和形参类型不一致的情况。
3. 	一般形式 
	>template \<class T1,...> 
	>class也可以用typename代替

```c++
#include <iostream>
template <class T> //末尾没有分号，class也可以写成typename
T max(T a,T b){
	return a;
}
int main()
{
	std::cout<<max(123,234);//调用时T换成int
	return 0;
}
```
