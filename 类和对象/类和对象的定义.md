## 一.类的定义

1. 定义一个类的语法格式
	```c++
	class NAME
	{
		//数据和成员函数
	};//注意末尾的分号
	```
	
2.	成员访问限定符
	+	**私有** *private* 成员只能被本类的成员函数和友元引用，类外不能调用。
	+	**公开** *public* 成员可以被内部和外界调用。
	+	**受保护的** *protected* 成员只能被本类的成员函数、友元、派生类的成员函数引用，其余位置不能调用。
	>[danger]如果在类的定义中既不指定private，也不指定public，则系统就默认为是私有的.
